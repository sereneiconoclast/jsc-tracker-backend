Steps to add a new method:

* Create the new handler, e.g. lib/post_user_user_id.rb
* Lambda stack: Add new function, e.g. JSCTrackerPostUserUserIdLambdaFunction, updating FunctionName and Handler
* Lambda stack: Add new output for new function ARN, e.g. JSCTrackerPostUserUserIdLambdaFunctionArn
* API Gateway stack: Add new resource(s) if the path is new
* API Gateway stack: Add new method, e.g. JSCTrackerPostUserUserIdMethod, updating ResourceId, HttpMethod,
  and LambdaFunctionArn
* API Gateway stack: Add new permission, e.g. JSCTrackerLambdaPermissionPostUserUserId, updating FunctionName
* API Gateway stack: Add new method to DependsOn under JSCTrackerDeployment
* Create-JscZip to upload code to S3
* Update-JscStack jsc-tracker-lambda
* Update-JscStack jsc-tracker-api-gateway
